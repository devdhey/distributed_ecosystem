# Explorando o MAPREDUCE
- O MapReduce é o modelo de programação do Hadoop para processamento distribuído de grandes volumes de dados. Ele divide as tarefas em duas fases principais: Map, onde os dados são processados e transformados, e Reduce, onde os resultados do Map são agregados para produzir a saída final.
- O MapReduce é altamente escalável e tolerante a falhas, permitindo que as tarefas sejam distribuídas por vários nós do cluster, e que o sistema se recupere automaticamente de falhas de hardware ou software.
